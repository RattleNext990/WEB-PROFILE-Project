# WEB-PROFILE-Project
🚀 Ghanzz Personal Portfolio
Selamat datang di repositori website portofolio pribadi milik Kautsar (Ghanzz)!

Website ini adalah representasi digital dari perjalananku sebagai seorang pelajar SMAN 5 Yogyakarta yang memiliki antusiasme tinggi terhadap dunia teknologi, pemrograman, dan pengembangan web. Melalui web ini, aku membagikan profil, proyek-proyek yang sedang/sudah kugerjakan, serta layanan kolaborasi yang aku tawarkan.

🌟 Tentang Website Ini
Portofolio ini dibangun menggunakan antarmuka single-page yang interaktif. Tujuannya adalah untuk memudahkan pengunjung mengenal diriku, melihat hasil karyaku, dan menghubungi untuk keperluan kerja sama, dengan tetap mempertahankan desain yang modern dan dinamis.

🛠️ Fitur & Bagian Utama
🏠 Home: Bagian perkenalan singkat (profil), passion di bidang teknologi, serta tautan langsung ke berbagai platform media sosial (Discord, GitHub, Facebook, Instagram).

🎬 My Projects: Menampilkan proyek karya unggulan. Saat ini menampilkan KARSA FILM, sebuah film pendek berdurasi lima menit karya siswa-siswi SMAN 5 Yogyakarta yang mengangkat tema pentingnya toleransi, harmonisasi, dan kerukunan antar umat beragama. Terintegrasi langsung dengan pemutar video YouTube.

🤝 Services: Daftar layanan dan proyek kolaborasi yang aku tawarkan, antara lain:

Fotografi

Video Editing

Animasi

Tools Robotik

Desain Poster

📫 Contact: Informasi kontak singkat, mencakup alamat email, media sosial, dan lokasi saat ini (Yogyakarta, Indonesia).

💻 Teknologi yang Digunakan
Website ini dikembangkan menggunakan teknologi front-end dasar:

HTML5: Sebagai kerangka dan struktur utama halaman.

CSS3: Untuk styling, tata letak, dan responsivitas (dihubungkan melalui Youtube.css).

JavaScript (Vanilla): Memberikan sentuhan interaktif seperti header yang menyembunyi secara otomatis saat di-scroll, serta efek parallax pada elemen-elemen yang mengambang.

FontAwesome: Digunakan untuk menampilkan ikon-ikon media sosial yang rapi.

🚀 Cara Menjalankan Secara Lokal
Untuk melihat website ini di komputermu sendiri, ikuti langkah-langkah mudah berikut:

Unduh/Clone repositori ini ke dalam perangkatmu.

Pastikan semua file inti (Youtube.html, Youtube.css) dan aset visual (seperti IMG_9025.JPG, logo/ikon) berada di dalam struktur folder yang sama.

Klik dua kali atau buka file Youtube.html menggunakan web browser andalanmu (Google Chrome, Firefox, Safari, dll). Tidak perlu instalasi server lokal!
